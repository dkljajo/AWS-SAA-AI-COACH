# AWS-SAA-AI-COACH

🧠 Tagline

AI-powered AWS learning platform that teaches cloud architecture through real-world scenarios, not memorization.

📖 STORY (USE FOR DEV.TO + GITHUB HERO SECTION)

While preparing for the AWS Solutions Architect Associate (SAA-C03) exam, I noticed a recurring problem:

Most learning resources focus heavily on memorization — services, features, limits — but the actual exam doesn’t test that.

It tests how you think.

It tests your ability to:

design architectures under constraints
evaluate trade-offs (cost, scalability, reliability)
combine multiple AWS services effectively

That gap is what led me to build this project.

🎬 THE IDEA

Instead of another quiz tool, I wanted to build something closer to how real learning should feel:

A Netflix-style AWS learning experience.

👉 Learn → Practice → Advanced Scenarios → Exam Mode

Each step progressively trains architectural thinking instead of memorization.

⚙️ HOW IT WORKS (USE IN BOTH DEV.TO + README)

Users go through a structured learning flow:

1. 📚 Learning Series
Explains AWS concepts simply
Provides guided examples
Builds foundational understanding
2. 🟢 Beginner Mode
Simple AWS scenarios
1–2 services per question
3. 🟡 Intermediate Mode
Multi-service architecture problems
Trade-off-based decisions
4. 🔴 Advanced Mode
Real exam-level complexity
Distractor-heavy questions
5. 🧪 Exam Mode (65 Questions)
Full SAA simulation
Mixed AWS domains
Real test pressure
🧠 WHAT MAKES IT DIFFERENT

Most AWS learning tools focus on:

This project focuses on:
